This file is presented as a JSON array, which is composed of a list of queries. 
Each query contains: 
1) A query string in LaTeX; 
2) A list of labels. Each label contains a URL and its score. 

====Definition of score====
score = {1, 2, 3, 4, 5}
1: No relevant to the query;
2-5: The higher score denotes the more relevant to the query.
